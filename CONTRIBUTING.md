# Contributing

Pull requests are welcome.

## How to contribute

1. Fork the repo
2. Create a branch
3. Make changes in `src/` (`neiki-forms.js`, `neiki-forms.css`)
4. Run `npm test` and `npm run build` to verify the build still works
5. Submit a PR

Please review the [Code of Conduct](CODE_OF_CONDUCT.md) before opening an issue or pull request. Security-related reports should follow [SECURITY.md](SECURITY.md) instead of a public issue.
